# Claude Code 슬래시 명령어

## 세션 관리

| 명령어 | 설명 |
|--------|------|
| `/clear` | 대화 기록 초기화 |
| `/rename [이름]` | 현재 세션 이름 변경 |
| `/resume` | 이전 대화 재개 |
| `/branch [이름]` | 현재 지점에서 대화 분기 |
| `/compact [지시사항]` | 대화 내용 압축 (컨텍스트 절약) |
| `/exit` | CLI 종료 |

## 코드 작업

| 명령어 | 설명 |
|--------|------|
| `/diff` | 미커밋 변경사항 diff 뷰어 |
| `/rewind` | 대화/코드를 이전 상태로 되돌리기 |
| `/pr-comments [PR번호]` | GitHub PR 댓글 가져오기 |
| `/security-review` | 변경사항 보안 취약점 분석 |
| `/add-dir <경로>` | 세션에 작업 디렉토리 추가 |

## 모델 / 성능

| 명령어 | 설명 |
|--------|------|
| `/model [모델명]` | 사용할 AI 모델 변경 |
| `/effort [low\|medium\|high\|max\|auto]` | 모델 노력도 설정 |
| `/fast` | 빠른 모드 토글 |
| `/plan` | 플랜 모드로 진입 |
| `/sandbox` | 샌드박스 모드 토글 |

## 설정

| 명령어 | 설명 |
|--------|------|
| `/config` | 설정 인터페이스 열기 |
| `/permissions` | 도구 권한 보기/수정 |
| `/theme` | 색상 테마 변경 |
| `/color [색상]` | 프롬프트 바 색상 설정 |
| `/keybindings` | 키바인딩 설정 |
| `/terminal-setup` | 터미널 키바인딩 구성 |
| `/memory` | CLAUDE.md 메모리 파일 편집 |
| `/init` | 프로젝트 CLAUDE.md 초기화 |

## MCP / 통합

| 명령어 | 설명 |
|--------|------|
| `/mcp` | MCP 서버 연결 관리 |
| `/hooks` | 훅 설정 보기 |
| `/ide` | IDE 통합 관리 |
| `/agents` | 서브에이전트 구성 관리 |
| `/skills` | 사용 가능한 skills 목록 |
| `/plugin` | Claude Code 플러그인 관리 |
| `/reload-plugins` | 모든 활성 플러그인 다시 로드 |

## 정보 / 유틸

| 명령어 | 설명 |
|--------|------|
| `/help` | 도움말 표시 |
| `/status` | 버전/모델/계정 상태 확인 |
| `/cost` | 토큰 사용량 확인 |
| `/stats` | 일일 사용량 시각화 |
| `/context` | 현재 컨텍스트 사용량 시각화 |
| `/insights` | 세션 분석 보고서 생성 |
| `/doctor` | 설치/설정 진단 |
| `/release-notes` | 전체 changelog 보기 |
| `/export [파일명]` | 대화 내용 텍스트로 내보내기 |
| `/copy [N]` | 최근 응답을 클립보드에 복사 |
| `/feedback` | 피드백/버그 제출 |

## 계정
